O que é HTML?

O HTML foi criado por TIM BERNERS-LEE.

O HTML Permite inserir o conteudo e estabelecer a estrutura basica de um website, ele serve para dar siguinificado e organizar as informações de uma pagina web, sem isso o navegador nao saberia exibir textos como elementos ou carregar imagens e outros conteudo.

O que são TAGS?

As tags HTML, são utilizadas para informar o navegador que tipo de estrutura esta sendo construida.

Digamos que voce quera escrever um paragrafo para isso devemos utilizar a tag <p>, tambem podemos utilizar tags para determinar titulos, imagens, linques, entre outros.

O que são ANCORAS

São links criados para levar para partes especificas da pagina.

Como é uma estrutura básica de um HTML?

Um documento HTML precisa seguir a estrutura composta pelas tags <html>, <head>, e <body>, e a instrução <DOCTYPE>.

O que são tabelas?

Uma tabela é um conjunto de dados estruturados composto de linhas e colunas.

As tabelas são definidas com a tag <TABLE> e divididas em linhas com a tag <tr>, cada linha é dividida em celulas de dados <td>.
